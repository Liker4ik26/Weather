## How to run

* You need Android Studio (at least Arctic Fox) with Kotlin plugin (at least 1.6.10)S
* Add `apiKey=<YourKey>` in `local.properties` file, for
  example `apiKey="72288ebc5b893cc92a76c701c6b700b6"`
* Run -> Run 'app'
